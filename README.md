# КосмоСуши — сайт доставки

Многофилиальный сервис доставки суши и роллов с витриной для клиентов и админ-панелью для управления заказами, складом и финансами.

---

## Для клиентов

### Главная и меню
- **Главная** (`/`) — баннеры, акции, популярные позиции, информация о доставке.
- **Меню** (`/menu`) — каталог по категориям: роллы, сеты, супы, напитки и т.д. Фильтры, поиск, карточки с фото, описанием, составом и калорийностью.
- **Карточка товара** — выбор модификаторов (соусы, добавки), количество, добавление в корзину.
- **Избранное** — сохранение любимых позиций (доступно после входа).

### Корзина и оформление заказа
- **Корзина** (выезжает справа) — список позиций, изменение количества, применение промокода, расчёт бонусов и итоговой суммы.
- **Оформление** (`/checkout`) — выбор способа получения (доставка / самовывоз), адрес с автоподсказками и проверкой зоны доставки, выбор времени (как можно скорее или на конкретное время), способ оплаты (карта при получении, наличные, онлайн), комментарий курьеру, использование бонусов.
- **Подтверждение** (`/order-success`) — номер заказа, статус, кнопка повторного заказа.

### Личный кабинет
- **Профиль** (`/account`) — данные пользователя, история заказов, бонусный баланс и уровень лояльности (бронза / серебро / золото), сохранённые адреса, реферальный код.
- **Вход / регистрация** (`/login`) — email + пароль, Google OAuth.

### Информационные страницы
- **Доставка** (`/delivery`) — зоны, минимальные суммы, время доставки.
- **О нас** (`/about`), **FAQ** (`/faq`), **Акции и новости** (`/news`).
- **Обратный звонок** — кнопка вызова формы с любой страницы.

### Бонусная программа
- Кэшбэк бонусами с каждого выполненного заказа.
- Уровни: бронза (от 0 ₽), серебро (от 10 000 ₽), золото (от 30 000 ₽) — каждый уровень даёт больший процент кэшбэка.
- Промокоды на скидку или подарочный товар.

## Админ-панель (`/admin`)

Доступ только пользователям с ролью `admin` или `super_admin`. Админ филиала видит только данные своего филиала, супер-админ — все филиалы.

### Разделы

**Дашборд** — ключевые метрики: выручка, количество заказов, средний чек, активные заказы.

**Канбан заказов** — заказы в виде колонок по статусам (новый → готовится → в доставке → доставлен). Перетаскивание между колонками, звуковое и push-уведомление о новых заказах.

**📞 Принять заказ (POS)** — ручное оформление заказа оператором: выбор товаров, клиент, адрес, оплата, печать чека.

**Заказы** — таблица всех заказов с фильтрами по дате, статусу, способу оплаты, источнику. Открытие карточки заказа, редактирование состава, печать чека и кухонной квитанции, оформление возврата.

**Заявки на звонок** — обращения с формы обратной связи.

**Клиенты** — база клиентов: история заказов, бонусы, уровень лояльности, ручная корректировка баланса.

**Отчёты** — выручка, прибыль, продажи по товарам и категориям за период.

**Кассовые смены** — открытие и закрытие смены, X- и Z-отчёты, контроль наличных в кассе. Чек по заказу можно пробить только при открытой смене.

**Склад по филиалам** — остатки ингредиентов и готовых позиций по каждому филиалу, перемещения между филиалами, инвентаризация.

**Прочие расходы** — учёт зарплат, аренды, коммуналки, закупок и др. с разбивкой по категориям и периоду.

**Ингредиенты / Техкарты** — справочник ингредиентов с себестоимостью, рецепты товаров (что и в каком количестве уходит при продаже), составные ингредиенты (полуфабрикаты).

**Модификаторы** *(только супер-админ)* — управление соусами, добавками и опциями к товарам.

**Накладные прихода** — оприходование закупок с автоматическим пересчётом средневзвешенной себестоимости.

**Списания и графики списаний** — ручное списание остатков и автоматические правила (например, обнуление готовой продукции в конце дня).

**Удалённые заказы** — корзина с возможностью восстановления.

**Филиалы** *(только супер-админ)* — управление точками: адрес, телефон, реквизиты онлайн-кассы (ОФД).

**Курьеры и зоны** *(только супер-админ)* — список курьеров и редактор зон доставки на карте с тарифами.

**Товары, Категории, Баннеры, Новости, Промокоды** *(только супер-админ)* — управление контентом витрины.

**Настройки чека** — реквизиты для кассового чека (название, ИНН оператора, место расчётов).

**Настройки** *(только супер-админ)* — общие настройки сайта.

### Роли
- **`super_admin`** — полный доступ ко всем филиалам и контенту. Единственный, кто может назначать роли.
- **`admin`** — управление одним филиалом: заказы, склад, расходы, смены, клиенты, отчёты по своему филиалу.

## Технологии

- **Frontend:** React 19, TanStack Start (SSR), TanStack Router, Tailwind CSS v4, shadcn/ui
- **Backend:** Lovable Cloud (PostgreSQL + RLS + Auth + Storage)
- **Платежи / касса:** интеграция с онлайн-ККТ через `kkt_url` филиала
- **Карты:** Google Maps для зон доставки и подсказок адреса
- **Email:** Resend для транзакционных писем

## Разработка

```bash
bun install
bun run dev
```

Откройте http://localhost:5173.

### Структура

- `src/routes/` — страницы (файловый роутинг TanStack)
- `src/components/` — переиспользуемые компоненты
- `src/lib/` — бизнес-логика, утилиты, серверные функции (`*.functions.ts`)
- `src/integrations/supabase/` — клиент БД (автогенерируется, не редактировать)
- `supabase/migrations/` — миграции схемы БД
